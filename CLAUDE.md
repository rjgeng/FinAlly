# FinAlly Project - the Finance Ally

All project documentation is in the `planning` directory.

## Repo operating rules

- `planning/PLAN.md` is the canonical, current implementation plan. Read it before making any planning or implementation decisions.
- `planning/archive/PLAN_v1.0.md` is historical reference only. Do not treat it as active requirements unless explicitly asked.
- When summarizing or acting on project requirements, rely on `planning/PLAN.md`, not archived versions.
- When updating the plan, edit `planning/PLAN.md` directly unless asked to create a new versioned draft.
